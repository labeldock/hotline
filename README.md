# hotline

[![Greenkeeper badge](https://badges.greenkeeper.io/labeldock/hotline.svg)](https://greenkeeper.io/)

## install
```
npm install
npm run init
```

## start
```
npm run start
```


## https mode
```
# make key.pem & cert.pem file instantly (only first time)
npm run makeHttpsCert

# run server
npm run https
```
